## 🌟 La Visione
**QUIP** nasce dalla necessità di centralizzare l'esperienza universitaria. Spesso i dati della carriera, gli appunti e le informazioni dell'ateneo sono sparsi su troppe piattaforme diverse. QUIP unisce tutto in un'unica interfaccia moderna, minimale e potenziata dall'AI.

## ✨ Feature Roadmap

### 🟢 Fase 1: Core MVP (In Sviluppo)
- [x] **Dashboard Carriera**: Gestione voti, calcolo media ponderata e aritmetica, conteggio CFU.
- [ ] **Simulatore Esami**: Funzione "Cosa succede se..." per prevedere la variazione della media.
- [ ] **Custom Themes**: Supporto nativo per temi colore personalizzabili e Dark Mode.

### 🟡 Fase 2: Gestione & Community
- [ ] **Mercato degli Appunti**: Piattaforma gratuita per l'upload e il download di materiale didattico.
- [ ] **Banca Domande**: Database condiviso delle domande poste durante gli esami.
- [ ] **Community**: Gruppi di studio locali per ogni ateneo.

### 🟠 Fase 3: AI & Integrazioni
- [ ] **AI Study Tutor**: Integrazione con Google Gemini per spiegazioni e pianificazione studio.
- [ ] **Ateneo Sync**: Collegamento automatico (dove possibile) o via crowdsourcing per orari e aule.
- [ ] **Mappa Universitaria**: Punti di interesse, biblioteche e sconti studenti nelle vicinanze.

## 🛠️ Tech Stack
- **Frontend:** [Flutter](https://flutter.dev) (Dart)
- **State Management:** [Provider](https://pub.dev/packages/provider)
- **Local Storage:** Shared Preferences / Isar DB
- **AI Integration:** Google AI Studio (Gemini API)

## 🚀 Installazione per Sviluppo

1. **Clona la repository:**
   ```bash
   git clone https://github.com/KurtSixxxx/quip.git
   ```
2. **Installa i pacchetti:**
   ```bash
   flutter pub get
   ```
3. **Esegui l'app:**
   ```bash
   flutter run
   ```
